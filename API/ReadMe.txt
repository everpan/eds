This package contains EDS Application Programming Interfaces C/C++ and Python.

Please refer to User's Manual for details on usage of this software.

Important notice:
The liveeds2 and archeds2 APIs have been deprecated in favor of the new edsapi.
Support for old APIs will be removed in 2013.

On linux:
Add to the start script or environment this line:
export LD_LIBRARY_PATH=$LD_LIBRARY_PATH:/usr/eds/API/C/lib

On Windows:
Check if this dll file exists in program directory!
